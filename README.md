## Overview
CLG automates licensing, certification, IMS/MAT compliance, and accreditation processes for behavioral health facilities.  
It integrates FastAPI, Sinatra-style avatars, and automated DHCS/Joint Commission monitoring.

---

## Key Features
- AI-driven compliance automation  
- FastAPI backend and modular folder structure  
- Avatar dashboard for P&P, QA/PI, Infection Control, Medication, Emergency Plan, and EOC  
- Automated GitHub version control and sync
